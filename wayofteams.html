Way of Teams

CTO Dashboard & AI Engineering Harness — Phoenix port

Way of Teams is a CTO Dashboard and AI Engineering Harness built on Phoenix, Ash Framework, and Jido agents. It replaces fragmented engineering management tools with a single, unified platform that combines tickets, standups, skills tracking, notifications, ideas, news, agent workflows, and observability — all in one real-time dashboard.

▼ (3) upsert_ticket() — Ash upsert

Component Reference

Layer File Role
Markdown thoughts/<project>/shared/tickets/ Authoritative ticket data (frontmatter + body)
Sync lib/wayofteams/tickets/sync.ex Pull f-rr-d, parse frontmatter, upsert to DB
DB lib/wayofteams/tickets/ticket.ex Ash Resource defining tickets table
Status lib/wayofteams/tickets/status.ex Lookup table: id → name + color
List UI live/tickets_live.ex Table with filters, bulk ops, sync trigger
Detail UI live/ticket_detail_live.ex Single ticket view + edit
Dashboard live/dashboard_live.ex Summary stat cards, recent tickets
My View live/my_view_live.ex Per-developer ticket list + standups
Kanban live/developers_live.ex Kanban board filtered by developer
Review live/review_queue_live.ex Tickets in review status
Create UI live/ticket_form_live.ex New ticket form (writes directly to DB)

Features

Tickets

Full ticket management with Ash resource lifecycle, status tracking, and GitHub integration. Synced from markdown files in f-rr-d.

Standups

Async standup management with per-developer daily updates aggregated into team view.

Skills & Insights

Skill matrix tracking with manifest system and automated SkillReport generation.

Agent Workflows

Jido-powered AI agents for automated signal processing, notifications, and workflow orchestration.

Ideas & News

Collaborative idea board and engineering news feed with Ash resources.

Observability

Full OpenTelemetry instrumentation across all layers — link team activity to system health.

Jido Agents

TicketManagerAgent

Ticket lifecycle management

SkillAutoUpdateAgent

Auto-discover and sync skills
